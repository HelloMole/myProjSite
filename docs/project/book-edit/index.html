项目描述

本项目由我在项目中发现开发流程过于复杂繁琐，为优化项目进度，针对交互式程序制作可自由编辑的编辑器，让策划无需会写代码也能实现动态绘本的制作，同时本编辑工具也适用于其它游戏场景。

项目业绩
将原本的开发难度降低，易用，降低了绘本的开发成本
详细内容

项目背景

在开发过程中发现交互式绘本的制作流程过于复杂，需要大量程序开发工作，导致开发成本高、周期长。

解决方案

开发了一个可视化编辑器，让非技术人员也能通过拖拽和配置的方式创建交互式内容。编辑器支持：

  • 可视化节点编辑
  • 丰富的交互组件库
  • 实时预览功能
  • 配置导出功能

技术亮点

  • 基于 Electron 开发的桌面应用
  • 支持多平台运行
  • 可扩展的插件系统
